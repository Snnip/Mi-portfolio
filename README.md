# Mi portfolio
This is my personal portfolio created with React Vite, where a little bit about me.

## Configuration
1. Clone this repository
2. Run npm install to install the dependencies.
3. Run npm run dev to start the development server.

## Technologies used
- React.js
- Vite
- React Router
- Tailwind CSS
